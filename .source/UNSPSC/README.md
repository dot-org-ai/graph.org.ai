# UNSPSC Data Sources

## Manual Download Required

UNSPSC (United Nations Standard Products and Services Code) requires registration.

1. Register at: https://www.unspsc.org/
2. Download the latest UNSPSC code set
3. Convert to TSV format with camelCase columns
4. Save as: UNSPSC.Codes.tsv

Expected columns:
- code
- title
- definition
- segmentCode
- segmentTitle
- familyCode
- familyTitle
- classCode
- classTitle
- commodityCode
- commodityTitle
