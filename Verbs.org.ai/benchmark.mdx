# Benchmark

Comparing organizational performance, processes, and practices against industry standards, competitors, or best-in-class organizations.

## APQC Process Context

Performance measurement and comparison activity that identifies performance gaps and improvement opportunities through comparison with external standards and peer organizations.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Performance.benchmark({ process, peer, metrics })
```

### Predicate (Relationship)
```typescript

$.Department.benchmarks.BestPractice
```

### Event (Past Tense)
```typescript
on($.Process.benchmarked, async (event) => {
  await $.Analysis.compare(event.metrics)
  await $.Strategy.identifyImprovements(event.gaps)
})
```

## APQC Process Category

Category 1: Plan & Organize
- Subcategory: Measure and analyze organizational performance
- Process: Conduct performance analysis and benchmarking

## Business Rules

- Benchmarking requires comparable data from peers
- Metrics must be clearly defined and measurable
- Comparison must account for contextual differences
- Gaps should drive improvement initiatives

## Related Verbs
- measure - Quantifying performance
- analyze - Examining data and results
- compare - Evaluating differences
- assess - Evaluating capability or performance
- improve - Making enhancements based on findings
