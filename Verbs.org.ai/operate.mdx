# Operate

Controlling and directing the functioning of machinery, equipment, systems, or organizations to achieve desired outcomes.

## O*NET Context

Work activity focused on running, controlling, and directing the performance of machines, equipment, vehicles, or complex systems to accomplish work tasks or produce results.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Equipment.operate({ mode: 'production', parameters: { speed: 100 } })
```

### Predicate (Relationship)
```typescript
$.Operator.operates.Machinery
$.Technician.operates.System
```

### Event (Past Tense)
```typescript
on($.Equipment.operated, async (event) => {
  await $.Log.record({ equipment: event.equipmentId, duration: event.duration })
  await $.Maintenance.schedule({ basedOn: 'usage' })
})
```

### Activity (Ongoing Process)
```typescript
const activeOperation = $.Factory.operating.AssemblyLine
console.log(`Currently operating: ${activeOperation.equipment}`)
```

### Role (Actor)
```typescript
const operator: $.Operator = await $.findAgent({ license: 'heavy-machinery' })
await operator.runCycle({ duration: '8-hours' })
```

### Concept (Abstraction)
```typescript
const operation: $.Operation = {
  operatorId: operator.id,
  equipmentId: equipment.id,
  startTime: now(),

  actualOutput: 0,

```

### Inverse (Passive)
```typescript

```
