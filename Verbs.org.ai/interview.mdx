# Interview

Conducting formal conversations with individuals through structured questioning to gather information, assess qualifications, or understand perspectives.

## O*NET Context

Work activity focused on engaging in one-on-one or small group conversations to collect data, evaluate candidates, understand needs, or gain insights into experiences or knowledge.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Recruiter.interview({ candidate: person, position: role, format: 'behavioral' })
```

### Predicate (Relationship)
```typescript
$.Interviewer.interviews.Candidate
$.Researcher.interviews.Subject
```

### Event (Past Tense)
```typescript
on($.Candidate.interviewed, async (event) => {
  await $.Feedback.collect({ interviewId: event.id })
  await $.Assessment.score({ candidateId: event.candidateId })
})
```

### Activity (Ongoing Process)
```typescript
const activeInterviewing = $.Team.interviewing.Applicants
console.log(`Candidates reviewed: ${activeInterviewing.count}`)
```

### Role (Actor)
```typescript
const interviewer: $.Interviewer = await $.findAgent({ expertise: 'talent-assessment' })
interviewer.evaluateCandidate(applicant)
```

### Concept (Abstraction)
```typescript
const interview: $.Interview = {
  interviewerId: interviewer.id,
  subjectId: candidate.id,
  questions: script,
  startTime: when,
  duration: minutes,
  rating: score
}
```

### Inverse (Passive)
```typescript
const candidates = await db.related($.Job, $.interviewedBy, $.Recruiter)
```
