# Present

Giving information or displaying something to an audience in a prepared, organized manner.

## O*NET Context

Work activity focused on communicating information, ideas, or findings to audiences through speeches, demonstrations, or visual displays.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Presenter.present({ topic: 'findings', audience: 'executives', format: 'slideshow' })
```

### Predicate (Relationship)
```typescript
$.Analyst.presents.Findings
$.Executive.presents.Strategy
```

### Event (Past Tense)
```typescript
on($.Proposal.presented, async (event) => {
  await $.Stakeholder.gather({ feedback: true })
  await $.Decision.schedule({ timeline: '1-week' })
})
```

### Activity (Ongoing Process)
```typescript
const activePresentation = $.Speaker.presenting.Webinar
console.log(`Currently presenting to ${activePresentation.attendeeCount} participants`)
```

### Role (Actor)
```typescript
const presenter: $.Presenter = await $.findAgent({ expertise: 'public-speaking' })
await presenter.deliverPresentation(slides)
```

### Concept (Abstraction)
```typescript
const presentation: $.Presentation = {
  subjectId: topic.id,
  title: '',
  content: [],
  audience: { type: '', size: 0 },
  format: 'slideshow',
  duration: 0,
  objectives: []
}
```

### Inverse (Passive)
```typescript
const speakers = await db.related($.Findings, $.presentedBy, $.Speaker)
```
