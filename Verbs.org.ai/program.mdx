# Program

Writing computer code, creating instructions, or scheduling operations and events to control systems or perform specific functions.

## O*NET Context

Work activity focused on writing software code, configuring systems with code, or scheduling activities and operations.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Developer.program({ language: 'typescript', functionality: 'authentication', testing: 'comprehensive' })
```

### Predicate (Relationship)
```typescript

```

### Event (Past Tense)

### Activity (Ongoing Process)
