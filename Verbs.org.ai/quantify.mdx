# Quantify

Measuring, counting, or expressing something in numerical, mathematical, or measurable terms to determine its amount, size, or extent.

## O*NET Context

Work activity focused on assigning numerical values, measurements, or quantities to data, materials, results, or outcomes for analysis, reporting, or decision-making purposes.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Analysis.quantify({ data: dataset, metrics: ['cost', 'impact', 'risk'] })
```

### Predicate (Relationship)
```typescript
$.Analyst.quantifies.Impact
$.Auditor.quantifies.Variance
$.Engineer.quantifies.Defects
```

### Event (Past Tense)
```typescript
on($.Measurement.quantified, async (event) => {
  await $.Report.include({ metric: event.value })
  await $.Database.store({ quantification: event })
})
```

### Activity (Ongoing Process)
```typescript
const activeQuantification = $.Team.quantifying.ResourceUsage
console.log(`Quantifying: ${activeQuantification.completionPercent}%`)
```

### Role (Actor)
```typescript
const quantifier: $.Quantifier = await $.findAgent({ expertise: 'data-analysis' })
await quantifier.measureValue(attribute)
```

### Concept (Abstraction)
```typescript
const quantification: $.Quantification = {
  subject: 'performance-improvement',
  metric: 'percentage-increase',
  baselineValue: 100,
  measuredValue: 125,
  increase: 25,
  unit: 'percent',
  confidence: 0.95
}
```

### Inverse (Passive)
```typescript
const measurements = await db.related($.Outcome, $.quantifiedBy, $.Analyst)
```
