# Pick

Selecting items from inventory for fulfillment or use.

## GS1 Context

Business step for formally selecting items from warehouse inventory in response to orders or other requests. This is a critical operation in fulfillment workflows where items are selected from various locations in the warehouse.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Order.pick({ itemIds: [item1, item2], timestamp: now(), location: 'Aisle-C' })
```

### Predicate (Relationship)
```typescript
$.Warehouse.picks.Inventory

```

### Event (Past Tense)
```typescript
on($.Order.picked, async (event) => {
  await $.QualityCheck.queue({ orderId: event.orderId, items: event.items })
  if (event.allItemsAvailable) await $.Packing.schedule({ orderId: event.orderId })
})
```

## Usage Examples

**Order Fulfillment:**
- Picking customer order items from warehouse
- Picking items for retail shelf replenishment
- Picking components for manufacturing orders
- Picking items for returns processing

**Business Rules:**
- Picking must follow order verification and availability check
- Items must be selected from correct warehouse locations
- Picked items must match order specifications exactly
- Picking completion updates inventory allocations
