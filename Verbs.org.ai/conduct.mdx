# Conduct

Directing, leading, managing, or carrying out activities, events, operations, or investigations in a structured and systematic manner.

## O*NET Context

Work activity focused on leading, managing, or directing work activities, meetings, interviews, or operations to achieve specific objectives.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Meeting.conduct({ participants: [user1, user2], duration: 60, agenda: topics })
```

### Predicate (Relationship)
```typescript
$.Manager.conducts.Meeting
$.Researcher.conducts.Study
```

### Event (Past Tense)
```typescript
on($.Investigation.conducted, async (event) => {
  await $.Report.generate({ findings: event.results })
  await $.Stakeholder.notify({ conclusion: event.outcome })
})
```

### Activity (Ongoing Process)
```typescript
const activeOperation = $.Team.conducting.Research
console.log(`Currently conducting: ${activeOperation.title}`)
```

### Role (Actor)
```typescript

conductor.manageMeeting(meetingDetails)
```

### Concept (Abstraction)

**Research and Investigation:**

- Managing research studies
- Investigating complaints or issues
- Auditing operations

**Operations Management:**
- Managing manufacturing processes
- Directing project execution
- Overseeing team activities
- Controlling workflow execution
