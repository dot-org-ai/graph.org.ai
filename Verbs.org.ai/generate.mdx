# Generate

Producing or creating something through a process or action, whether physical output, data, ideas, or value.

## O*NET Context

Fundamental work activity across numerous occupations including manufacturers, software developers, artists, and scientists. Encompasses both creative generation (ideas, designs) and productive generation (goods, energy, data).

## Semantic Forms

### Action (Base Form)
```typescript
await $.Power.generate({ type: 'solar', capacity: '10MW' })
```

## Usage Examples

**Manufacturing:**
- Factories generate products from raw materials
- Machines generate precise components
- Production lines generate consistent output

**Technology:**
- Systems generate reports automatically
- Algorithms generate recommendations
- AI models generate text or images

**Energy/Resource:**
- Power plants generate electricity
- Wells generate oil or natural gas
- Processes generate heat or waste products

**Business Rules:**
- Generation tracked with timestamp and source
- Quality metrics recorded for all generation
- Capacity constraints enforced
- Waste or byproducts documented
