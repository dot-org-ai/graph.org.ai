# Perform

Executing, carrying out, or presenting an action or task to achieve a desired result or complete an objective.

## O*NET Context

Work activity focused on executing work tasks, carrying out responsibilities, or presenting services to accomplish objectives and meet organizational or customer requirements.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Task.perform({ priority: 'high', deadline: 'urgent' })
```

### Predicate (Relationship)
```typescript
$.Employee.performs.Duty
$.Technician.performs.Service
```

### Event (Past Tense)
```typescript
on($.Task.performed, async (event) => {
  await $.Quality.validate({ output: event.result })
  await $.Supervisor.verify({ completion: event.timestamp })
})
```

### Activity (Ongoing Process)
```typescript
const activePerformance = $.Team.performing.MaintenanceWork
console.log(`Currently performing: ${activePerformance.task}`)
```

### Role (Actor)
```typescript
const performer: $.Performer = await $.findAgent({ skillLevel: 'expert' })
await performer.executeTask({ task, standards: 'ISO-certified' })
```

### Concept (Abstraction)
```typescript
const performance: $.Performance = {
  performerId: performer.id,
  taskId: task.id,
  startTime: now(),
  completionTime: null,
  qualityRating: 0,
  metrics: {}
}
```

### Inverse (Passive)
```typescript
const completions = await db.related($.Service, $.performedBy, $.Performer)
```
