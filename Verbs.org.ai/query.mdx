# Query

Asking questions or making requests for specific information, data, or clarification from databases, systems, or other sources.

## O*NET Context

Work activity focused on requesting information through questions, database searches, or inquiry processes to obtain answers, data, or clarification needed for tasks or decisions.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Database.query({ table: 'customers', conditions: { status: 'active' } })
```

### Predicate (Relationship)
```typescript
$.Analyst.queries.Database
$.Support.queries.KnowledgeBase
$.Researcher.queries.Archive
```

### Event (Past Tense)
```typescript
on($.Request.queried, async (event) => {
  await $.System.processResults({ query: event.id })
  await $.Log.record({ queryTime: event.duration })
})
```

### Activity (Ongoing Process)
```typescript
const activeQuery = $.System.querying.LargeDataset
console.log(`Query progress: ${activeQuery.completionPercent}%`)
```

### Role (Actor)
```typescript

await querier.searchRecords(criteria)
```

### Concept (Abstraction)
