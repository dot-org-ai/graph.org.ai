# Recommend

Suggesting or endorsing a particular course of action, solution, product, or approach based on analysis, experience, or professional judgment.

## O*NET Context

Work activity focused on providing professional suggestions or endorsements based on expertise, analysis, or best practices to guide decision-making or actions.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Advisor.recommend({ option: solution, basis: 'cost-benefit-analysis', confidence: 0.92 })
```

### Predicate (Relationship)
```typescript
$.Expert.recommends.Approach
$.Consultant.recommends.Strategy
$.Doctor.recommends.Treatment
```

### Event (Past Tense)
```typescript
on($.Suggestion.recommended, async (event) => {
  await $.Decision.consider({ recommendation: event.id })
  await $.Stakeholder.review({ proposal: event.details })
})
```

### Activity (Ongoing Process)
```typescript
const activeRecommendation = $.Team.recommending.Solutions
console.log(`Evaluating: ${activeRecommendation.candidateCount} options`)
```

### Role (Actor)
```typescript
const recommender: $.Recommender = await $.findAgent({ expertise: 'strategic-advisory' })
await recommender.endorseApproach(option)
```

### Concept (Abstraction)
```typescript
const recommendation: $.Recommendation = {
  description: 'Implement cloud migration strategy',
  rationale: 'Improved scalability and cost efficiency',
  priority: 'high',
  expectedOutcome: '30% cost reduction',
  timeline: '6 months',
  recommendedBy: advisorId,
  confidence: 0.88
}
```

### Inverse (Passive)
```typescript
const advisors = await db.related($.Solution, $.recommendedBy, $.Expert)
```
