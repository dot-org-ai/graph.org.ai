# Produce

Making or generating something, whether through manufacturing, creation, preparation, or generation of goods or outputs.

## O*NET Context

Work activity focused on manufacturing goods, creating content, generating reports, or producing deliverables through organized effort.

## Semantic Forms

### Action (Base Form)
```typescript
await $.Facility.produce({ product: 'widgets', quantity: 1000, timeline: 'monthly' })
```

### Predicate (Relationship)
```typescript
$.Company.produces.Goods
$.Team.produces.Content
```

### Event (Past Tense)
```typescript
on($.Goods.produced, async (event) => {
  await $.Quality.inspect({ items: event.quantity })
  await $.Shipping.prepare({ destination: event.destination })
})
```

### Activity (Ongoing Process)
```typescript
const activeProduction = $.Facility.producing.Inventory
console.log(`Currently producing: ${activeProduction.outputPerHour} units/hour`)
```

### Role (Actor)
```typescript
const producer: $.Producer = await $.findAgent({ expertise: 'manufacturing' })
await producer.createOutput(specifications)
```

### Concept (Abstraction)
```typescript
const production: $.Production = {
  subjectId: facility.id,
  productType: '',
  quantity: 0,
  timeline: { start: date, end: date },
  resources: [],
  costPerUnit: 0,
  outputQuality: {}
}
```

### Inverse (Passive)
```typescript
const facilities = await db.related($.Goods, $.producedBy, $.Facility)
```
