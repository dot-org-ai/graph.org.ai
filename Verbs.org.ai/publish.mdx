# Publish

Making content, information, research, or material publicly available through print, digital media, or other distribution channels.

## O*NET Context

Work activity focused on preparing, processing, and distributing written, visual, or multimedia content for public consumption, including books, articles, reports, software, and digital media.

## Semantic Forms

### Action (Base Form)
```typescript
await $.ContentTeam.publish({ article: manuscript, platforms: ['web', 'print', 'social'] })
```

### Predicate (Relationship)
```typescript
$.Author.publishes.Article
$.Studio.publishes.Music
$.Press.publishes.Book
```

### Event (Past Tense)
```typescript
on($.Document.published, async (event) => {
  await $.Audience.access({ publication: event.id })
  await $.Analytics.track({ event: 'publication-released' })
})
```

### Activity (Ongoing Process)
```typescript
const activePublishing = $.Editor.publishing.ResearchPaper
console.log(`Publishing status: ${activePublishing.completionPercent}%`)
```

### Role (Actor)
```typescript
const publisher: $.Publisher = await $.findAgent({ expertise: 'content-distribution' })
await publisher.releaseContent(manuscript)
```

### Concept (Abstraction)
```typescript
const publication: $.Publication = {
  title: 'Annual Report 2024',
  format: ['PDF', 'HTML', 'Print'],
  author: authorId,
  releaseDate: date,
  audience: 'public',
  channels: ['website', 'social-media', 'print'],
  distribution: 50000
}
```

### Inverse (Passive)
```typescript
const creators = await db.related($.Article, $.publishedBy, $.Author)
```
