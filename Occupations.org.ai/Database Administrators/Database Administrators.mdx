# Database Administrators

**O*NET-SOC Code**: 15-1242.00

Administer, test, and implement computer databases, applying knowledge of database management systems. Coordinate changes to computer databases. Identify, investigate, and resolve database performance issues, database capacity, and database scalability. May plan, coordinate, and implement security measures to safeguard computer databases.

## Tasks

- [Modify existing databases and database management systems or direct programmers and analysts to make changes.](Tasks/1299.mdx)
- [Plan, coordinate, and implement security measures to safeguard information in computer files against accidental or unauthorized damage, modification or disclosure.](Tasks/1301.mdx)
- [Plan and install upgrades of database management system software to enhance database performance.](Tasks/21649.mdx)
- [Specify users and user access levels for each segment of database.](Tasks/1305.mdx)
- [Test changes to database applications or systems.](Tasks/21651.mdx)
- [Test programs or databases, correct errors, and make necessary modifications.](Tasks/1300.mdx)
- [Train users and answer questions.](Tasks/1303.mdx)
- [Provide technical support to junior staff or clients.](Tasks/21650.mdx)
- [Approve, schedule, plan, and supervise the installation and testing of new products and improvements to computer systems, such as the installation of new databases.](Tasks/1302.mdx)
- [Develop standards and guidelines for the use and acquisition of software and to protect vulnerable information.](Tasks/1298.mdx)
