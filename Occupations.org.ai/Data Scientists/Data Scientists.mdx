# Data Scientists

**O*NET-SOC Code**: 15-2051.00

Develop and implement a set of techniques or analytics applications to transform raw data into meaningful information using data-oriented programming languages and visualization software. Apply data mining, data modeling, natural language processing, and machine learning to extract and analyze information from large structured and unstructured datasets. Visualize, interpret, and report data findings. May create dynamic data reports.

## Tasks

- [Analyze, manipulate, or process large sets of data using statistical software.](Tasks/21823.mdx)
- [Apply feature selection algorithms to models predicting outcomes of interest, such as sales, attrition, and healthcare use.](Tasks/21824.mdx)
- [Apply sampling techniques to determine groups to be surveyed or use complete enumeration methods.](Tasks/21825.mdx)
- [Clean and manipulate raw data using statistical software.](Tasks/21826.mdx)
- [Compare models using statistical performance metrics, such as loss functions or proportion of explained variance.](Tasks/21827.mdx)
- [Create graphs, charts, or other visualizations to convey the results of data analysis using specialized software.](Tasks/21828.mdx)
- [Deliver oral or written presentations of the results of mathematical modeling and data analysis to management or other end users.](Tasks/21829.mdx)
- [Design surveys, opinion polls, or other instruments to collect data.](Tasks/21830.mdx)
- [Identify business problems or management objectives that can be addressed through data analysis.](Tasks/21831.mdx)
- [Identify relationships and trends or any factors that could affect the results of research.](Tasks/21832.mdx)
