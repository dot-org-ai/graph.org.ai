# Computer Network Support Specialists

**O*NET-SOC Code**: 15-1231.00

Analyze, test, troubleshoot, and evaluate existing network systems, such as local area networks (LAN), wide area networks (WAN), cloud networks, servers, and other data communications networks. Perform network maintenance to ensure networks operate correctly with minimal interruption.

## Tasks

- [Back up network data.](Tasks/19004.mdx)
- [Configure security settings or access permissions for groups or individuals.](Tasks/18991.mdx)
- [Analyze and report computer network security breaches or attempted breaches.](Tasks/19003.mdx)
- [Identify the causes of networking problems, using diagnostic testing software and equipment.](Tasks/18995.mdx)
- [Document network support activities.](Tasks/18993.mdx)
- [Configure wide area network (WAN) or local area network (LAN) routers or related equipment.](Tasks/18992.mdx)
- [Install network software, including security or firewall software.](Tasks/18997.mdx)
- [Troubleshoot network or connectivity problems for users or user groups.](Tasks/19002.mdx)
- [Provide telephone support related to networking or connectivity issues.](Tasks/19010.mdx)
- [Evaluate local area network (LAN) or wide area network (WAN) performance data to ensure sufficient availability or speed, to identify network problems, or for disaster recovery purposes.](Tasks/18994.mdx)
