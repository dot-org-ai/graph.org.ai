# Information Security Engineers

**O*NET-SOC Code**: 15-1299.05

Develop and oversee the implementation of information security procedures and policies. Build, maintain and upgrade security technology, such as firewalls, for the safe use of computer networks and the transmission and retrieval of information. Design and implement appropriate security controls to identify vulnerabilities and protect digital files and electronic infrastructures. Monitor and respond to computer security breaches, viruses, and intrusions, and perform forensic investigation. May oversee the assessment of information security systems.

## Tasks

- [Identify security system weaknesses, using penetration tests.](Tasks/21776.mdx)
- [Coordinate monitoring of networks or systems for security breaches or intrusions.](Tasks/21769.mdx)
- [Assess the quality of security controls, using performance indicators.](Tasks/21766.mdx)
- [Train staff on, and oversee the use of, information security standards, policies, and best practices.](Tasks/21783.mdx)
- [Scan networks, using vulnerability assessment tools to identify vulnerabilities.](Tasks/21782.mdx)
- [Develop response and recovery strategies for security breaches.](Tasks/21774.mdx)
- [Conduct investigations of information security breaches to identify vulnerabilities and evaluate the damage.](Tasks/21767.mdx)
- [Develop or install software, such as firewalls and data encryption programs, to protect sensitive information.](Tasks/21773.mdx)
- [Oversee development of plans to safeguard computer files against accidental or unauthorized modification, destruction, or disclosure or to meet emergency data processing needs.](Tasks/21777.mdx)
- [Identify or implement solutions to information security problems.](Tasks/21775.mdx)
