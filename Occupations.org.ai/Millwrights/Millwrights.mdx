# Millwrights

**O*NET-SOC Code**: 49-9044.00

Install, dismantle, or move machinery and heavy equipment according to layout plans, blueprints, or other drawings.

## Tasks

- [Replace defective parts of machine, or adjust clearances and alignment of moving parts.](Tasks/3041.mdx)
- [Align machines or equipment, using hoists, jacks, hand tools, squares, rules, micrometers, lasers, or plumb bobs.](Tasks/21083.mdx)
- [Insert shims, adjust tension on nuts and bolts, or position parts, using hand tools and measuring instruments, to set specified clearances between moving and stationary parts.](Tasks/3048.mdx)
- [Signal crane operator to lower basic assembly units to bedplate, and align unit to centerline.](Tasks/3047.mdx)
- [Conduct preventative maintenance and repair, and lubricate machines and equipment.](Tasks/18785.mdx)
- [Assemble and install equipment, using hand tools and power tools.](Tasks/3045.mdx)
- [Assemble machines, and bolt, weld, rivet, or otherwise fasten them to foundation or other structures, using hand tools and power tools.](Tasks/3051.mdx)
- [Move machinery and equipment, using hoists, dollies, rollers, and trucks.](Tasks/3049.mdx)
- [Level bedplate and establish centerline, using straightedge, levels, and transit.](Tasks/3054.mdx)
- [Dismantle machines, using hammers, wrenches, crowbars, and other hand tools.](Tasks/3055.mdx)
