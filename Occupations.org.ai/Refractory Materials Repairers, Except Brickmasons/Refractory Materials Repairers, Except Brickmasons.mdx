# Refractory Materials Repairers, Except Brickmasons

**O*NET-SOC Code**: 49-9045.00

Build or repair equipment such as furnaces, kilns, cupolas, boilers, converters, ladles, soaking pits, and ovens, using refractory materials.

## Tasks

- [Reline or repair ladles and pouring spouts with refractory clay, using trowels.](Tasks/13819.mdx)
- [Chip slag from linings of ladles or remove linings when beyond repair, using hammers and chisels.](Tasks/13820.mdx)
- [Mix specified amounts of sand, clay, mortar powder, and water to form refractory clay or mortar, using shovels or mixing machines.](Tasks/13821.mdx)
- [Measure furnace walls to determine dimensions and cut required number of sheets from plastic block, using saws.](Tasks/13822.mdx)
- [Dry and bake new linings by placing inverted linings over burners, building fires in ladles, or by using blowtorches.](Tasks/13824.mdx)
- [Remove worn or damaged plastic block refractory linings of furnaces, using hand tools.](Tasks/13825.mdx)
- [Climb scaffolding, carrying hoses, and spray surfaces of cupolas with refractory mixtures, using spray equipment.](Tasks/13827.mdx)
- [Spread mortar on stopper heads and rods, using trowels, and slide brick sleeves over rods to form refractory jackets.](Tasks/13829.mdx)
- [Dump and tamp clay in molds, using tamping tools.](Tasks/13830.mdx)
- [Transfer clay structures to curing ovens, melting tanks, and drawing kilns, using forklifts.](Tasks/13832.mdx)
