# Data Warehousing Specialists

**O*NET-SOC Code**: 15-1243.01

Design, model, or implement corporate data warehousing activities. Program and configure warehouses of database information and provide support to warehouse users.

## Tasks

- [Develop data warehouse process models, including sourcing, loading, transformation, and extraction.](Tasks/16132.mdx)
- [Verify the structure, accuracy, or quality of warehouse data.](Tasks/16121.mdx)
- [Map data between source systems, data warehouses, and data marts.](Tasks/16124.mdx)
- [Develop and implement data extraction procedures from other systems, such as administration, billing, or claims.](Tasks/16126.mdx)
- [Design and implement warehouse database structures.](Tasks/16128.mdx)
- [Develop or maintain standards, such as organization, structure, or nomenclature, for the design of data warehouse elements, such as data architectures, models, tools, and databases.](Tasks/16127.mdx)
- [Provide or coordinate troubleshooting support for data warehouses.](Tasks/16118.mdx)
- [Write new programs or modify existing programs to meet customer requirements, using current programming languages and technologies.](Tasks/16120.mdx)
- [Design, implement, or operate comprehensive data warehouse systems to balance optimization of data access with batch loading and resource utilization factors, according to customer requirements.](Tasks/16133.mdx)
- [Perform system analysis, data analysis or programming, using a variety of computer languages and procedures.](Tasks/16123.mdx)
