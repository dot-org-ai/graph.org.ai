# Web Administrators

**O*NET-SOC Code**: 15-1299.01

Manage web environment design, deployment, development and maintenance activities. Perform testing and quality assurance of web sites and web applications.

## Tasks

- [Monitor systems for intrusions or denial of service attacks, and report security breaches to appropriate personnel.](Tasks/14734.mdx)
- [Identify or document backup or recovery plans.](Tasks/14741.mdx)
- [Back up or modify applications and related data to provide for disaster recovery.](Tasks/14731.mdx)
- [Correct testing-identified problems, or recommend actions for their resolution.](Tasks/14758.mdx)
- [Identify, standardize, and communicate levels of access and security.](Tasks/14748.mdx)
- [Determine sources of Web page or server problems, and take action to correct such problems.](Tasks/14732.mdx)
- [Implement updates, upgrades, and patches in a timely manner to limit loss of service.](Tasks/14740.mdx)
- [Implement Web site security measures, such as firewalls or message encryption.](Tasks/14735.mdx)
- [Collaborate with development teams to discuss, analyze, or resolve usability issues.](Tasks/14737.mdx)
- [Test issues such as system integration, performance, and system security on a regular schedule or after any major program modifications.](Tasks/14750.mdx)
