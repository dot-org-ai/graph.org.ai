# Security Management Specialists

**O*NET-SOC Code**: 13-1199.07

Conduct security assessments for organizations, and design security systems and processes. May specialize in areas such as physical security or the safety of employees and facilities.

## Tasks

- [Assess the nature and level of physical security threats so that the scope of the problem can be determined.](Tasks/21482.mdx)
- [Respond to emergency situations on an on-call basis.](Tasks/21501.mdx)
- [Recommend improvements in security systems or procedures.](Tasks/21500.mdx)
- [Perform risk analyses so that appropriate countermeasures can be developed.](Tasks/21496.mdx)
- [Inspect physical security design features, installations, or programs to ensure compliance with applicable standards or regulations.](Tasks/21491.mdx)
- [Conduct security audits to identify potential vulnerabilities related to physical security or staff safety.](Tasks/21484.mdx)
- [Design security policies, programs, or practices to ensure adequate security relating to alarm response, access card use, and other security needs.](Tasks/21485.mdx)
- [Test security measures for final acceptance and implement or provide procedures for ongoing monitoring and evaluation of the measures.](Tasks/21503.mdx)
- [Design, implement, or establish requirements for security systems, video surveillance, motion detection, or closed-circuit television systems to ensure proper installation and operation.](Tasks/21486.mdx)
- [Prepare, maintain, or update security procedures, security system drawings, or related documentation.](Tasks/21498.mdx)
