# Desktop Publishers

**O*NET-SOC Code**: 43-9031.00

Format typescript and graphic elements using computer software to produce publication-ready material.

- [Operate desktop publishing software and equipment to design, lay out, and produce camera-ready copy.](Tasks/2816.mdx)

- [Check preliminary and final proofs for errors and make necessary corrections.](Tasks/2815.mdx)
- [View monitors for visual representation of work in progress and for instructions and feedback throughout process, making modifications as necessary.](Tasks/2817.mdx)
- [Enter text into computer keyboard and select the size and style of type, column width, and appropriate spacing for printed materials.](Tasks/2818.mdx)
