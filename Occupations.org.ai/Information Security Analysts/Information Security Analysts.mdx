# Information Security Analysts

**O*NET-SOC Code**: 15-1212.00

Plan, implement, upgrade, or monitor security measures for the protection of computer networks and information. Assess system vulnerabilities for security risks and propose and implement risk mitigation strategies. May ensure appropriate security controls are in place that will safeguard digital files and vital electronic infrastructure. May respond to computer security breaches and viruses.

## Tasks

- [Develop plans to safeguard computer files against accidental or unauthorized modification, destruction, or disclosure and to meet emergency data processing needs.](Tasks/5314.mdx)
- [Monitor current reports of computer viruses to determine when to update virus protection systems.](Tasks/5316.mdx)
- [Encrypt data transmissions and erect firewalls to conceal confidential information as it is being transmitted and to keep out tainted digital transfers.](Tasks/5321.mdx)
- [Perform risk assessments and execute tests of data processing system to ensure functioning of data processing activities and security measures.](Tasks/5320.mdx)
- [Modify computer security files to incorporate new software, correct errors, or change individual access status.](Tasks/5317.mdx)
- [Review violations of computer security procedures and discuss procedures with violators to ensure violations are not repeated.](Tasks/5323.mdx)
- [Document computer security and emergency measures policies, procedures, and tests.](Tasks/5322.mdx)
- [Confer with users to discuss issues such as computer data access needs, security violations, and programming changes.](Tasks/5315.mdx)
- [Monitor use of data files and regulate access to safeguard information in computer files.](Tasks/5319.mdx)
- [Coordinate implementation of computer system plan with establishment personnel and outside vendors.](Tasks/5318.mdx)
