# computer software (non games) - digital

**GPC Brick Code**: 10006236
**Class**: [computer/video game software](Computer—Video Game Software.mdx)
**Family**: [computers/video game](../Computers—Video Game.mdx)

Includes any products that can be described / observed as a programme which is downloaded or streamed that instructs a computer how to process data along with documentation that explains how these programs should be used. Includes products such as Word Processing Software, Spreadsheet Software, Presentation Software, Fonts, and Desktop Publishing Software. Excludes products such as Computer Games and Digital Mobile Apps.
