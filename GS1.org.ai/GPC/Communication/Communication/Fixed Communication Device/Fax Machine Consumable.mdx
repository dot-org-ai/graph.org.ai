# fax machine consumable

**GPC Brick Code**: 10005677
**Class**: [fixed communication device](Fixed Communication Device.mdx)
**Family**: [communication](../Communication.mdx)
**Segment**: [communication](../../Communication.mdx)

Includes any products that can be described/observed as being specifically designed to periodically replace/refill components of a fax machine that have been depleted or worn out by use, such as ink films. The products include Drum Cartridges, Film Cartridges and kits which contain multiple replacement items such as toner, drum, and imaging unit. Excludes products such as Fax Machines, Printer and Type Writer Consumables.
