# Solid state drive SSD

**UNSPSC Code**: 43201830
**Class**: [Media storage devices](Media storage devices.mdx)
**Family**: [Components for information technology or broadcasting or telecommunications](../Components for information technology or broadcasting or telecommunications.mdx)
**Segment**: [Information Technology Broadcasting and Telecommunications](../../Information Technology Broadcasting and Telecommunications.mdx)

A solid-state drive SSD is a data storage device that uses solid-state memory to store persistent data.An SSD emulates a hard disk drive interface, thus easily replacing it in most applications. An SSD using SRAM or DRAM instead of flash memory is often called a RAM drive.
