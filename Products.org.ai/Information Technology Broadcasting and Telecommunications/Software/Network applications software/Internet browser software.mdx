Internet browsing software is a program which allows a person to read hypertext. The browser gives some means of viewing the contents of pages and of navigating from one page to another. Netscape Navigator, NCSA Mosaic, Lynx, and W3 are examples for browsers for the World-Wide Web. They act as clients to remote web servers.
