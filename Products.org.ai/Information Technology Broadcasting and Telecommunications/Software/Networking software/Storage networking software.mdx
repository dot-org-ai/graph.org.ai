# Storage networking software

**UNSPSC Code**: 43232907
**Class**: [Networking software](Networking software.mdx)
**Family**: [Software](../Software.mdx)
**Segment**: [Information Technology Broadcasting and Telecommunications](../../Information Technology Broadcasting and Telecommunications.mdx)

Storage management software is a software that enables network administrators to route backup data from various devices on a network to another device such as a server or a magnetic tape backup unit. This is done either to make use of a high-capacity storage system such as a tape juke-box or for disaster protection.
