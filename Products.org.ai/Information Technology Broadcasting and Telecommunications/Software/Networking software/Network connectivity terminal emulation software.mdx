# Network connectivity terminal emulation software

**UNSPSC Code**: 43232911
**Class**: [Networking software](Networking software.mdx)
**Family**: [Software](../Software.mdx)
**Segment**: [Information Technology Broadcasting and Telecommunications](../../Information Technology Broadcasting and Telecommunications.mdx)

A program that allows a computer to act like a particular brand of terminal, e.g. a vt-100. The computer thus appears as a terminal to the host computer and accepts the same escape sequences for functions such as cursor positioning and clearing the screen. xterm is a terminal emulator for the X Window System.
